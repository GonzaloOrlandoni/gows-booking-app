# 🦷 Lumina Dental Clinic - Booking System

![Lumina Banner](https://images.unsplash.com/photo-1606811841689-23dfddce3e95?auto=format&fit=crop&q=80&w=1200)

## 🚀 Sobre el Proyecto

**Lumina Dental Clinic** es una plataforma de gestión de turnos online diseñada para clínicas odontológicas de alta gama. Este proyecto forma parte del portfolio de **GO Web Solutions**, demostrando capacidades avanzadas en el desarrollo de interfaces interactivas, flujos de usuario complejos y diseño UI/UX premium.

El sistema permite a los pacientes reservar citas de manera autónoma, seleccionando el servicio, el profesional y el horario disponible, todo en una experiencia fluida y animada.

## 🛠️ Tech Stack

- **React 19** + **Vite 6** (Frontend ultra rápido)
- **TailwindCSS v4** (Estilos modernos y optimizados)
- **Framer Motion** (Animaciones de interfaz y transiciones de pasos)
- **AOS (Animate On Scroll)** (Revelación dinámica de secciones)
- **Lucide React** (Iconografía minimalista)
- **Date-fns** (Gestión precisa de calendarios y fechas)
- **React Hot Toast** (Notificaciones de feedback en tiempo real)

## ✨ Características Destacadas

- ✅ **Flujo de Reserva por Pasos:** Interfaz guiada que reduce la fricción del usuario.
- ✅ **Simulación de Disponibilidad Real:** Animación de búsqueda que mejora la percepción de robustez del sistema.
- ✅ **Diseño Mobile-First:** Adaptabilidad total para que los pacientes reserven desde cualquier dispositivo.
- ✅ **Landing Page Integrada:** Secciones de Testimonios, FAQ, Galería y Estadísticas para generar confianza.
- ✅ **Botón de WhatsApp Directo:** Canal de comunicación inmediata para consultas rápidas.

## 📦 Instalación y Uso

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/gows-booking-app.git
   ```
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```
